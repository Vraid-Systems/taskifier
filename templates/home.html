{% extends 'base.html' %} {% block title %}usage{% endblock %} {% block content %} task_id - unsigned numeric identifier/key of task
owner_key - text string used in identifing task owner to this system

Store task data:
POST /data/[owner_key]/[task_id]
application/x-www-form-urlencoded payload with POST parameters: source, dest, content, ready_time. ready_time must be in ISO-8601 format (UTC) - 2013-09-13T04:22:58.578Z - it is the earliest time at which the task should be processed.

Retrieve task data as JSON in response body:
GET /data/public/[task_id]
GET /data/[owner_key]/[task_id]
Example JSON output: {"id": 343500, "source": "robot@app.domain", "dest": "user@other.domain", "content": "notification message text", "ready_time": "2013-09-13T04:22:58.578Z"}

Delete task data:
DELETE /data/[owner_key]/[task_id]
